Use the API from dummyjson

Document: https://dummyjson.com/docs/
Request:
- Create files in 'src/api/dummyjson.ts' for create module "fetchProducts" use async/await and fetch function to get API from: https://dummyjson.com/docs/products
- Import "fetchProducts" into 'src/index.ts' and call it to get list Products and display to ul#products-list
- Make style for products list: display as grid with 3 columns like this https://demo.templatesjungle.com/ministore/shop.html
- Update module "fetchProducts" for display 12 products (use https://dummyjson.com/docs/products#products-limit_skip). And when click button "Load more" will be display 12 products more
- Create module "searchProducts" use async/await and fetch function to get API from: https://dummyjson.com/docs/products#products-search to search products by name. When type in input search and click button "Search", the products list will be filter by name
- Create module "fetchProductsCategories" use async/await and fetch function to get API from: https://dummyjson.com/docs/products#products-categories to get list categories of products. Display list categories to ul#categories-list.
- Create module "fetchProductsByCategory" use async/await and fetch function to get API from: https://dummyjson.com/docs/products#products-category to get list products by category when click on category item above
- Update function loar-more when display products by category page and products search page
- Update API endpoint to match product sort functionality
