Use the API from jsonplaceholder

Document: https://jsonplaceholder.typicode.com/

Example 1: Get list posts

    Request:
    - Create files in 'src/api/jsonplaceholder.ts' for create module "fetchPosts" use async/await and fetch function to get API from: https://jsonplaceholder.typicode.com/posts
    - Import "fetchPosts" into 'src/index.ts' and call it to get list posts and display to ul#post-list
    - When click to the post title, the post body will be displayed below the title
    - Create and export module "fetchComments(postId)" in 'src/api/jsonplaceholder.ts' file to get comments of a post by postId: https://jsonplaceholder.typicode.com/comments?postId=${postId}
    - Import "fetchComments" into 'src/index.ts' and call it when click to the post title to get comments of the post and display to the post item

    Example 2: Get photo by album

    Request:
    - In 'src/api/jsonplaceholder.ts' file, create and export module "fetchAlbums" and "fetchPhotos" use async/await and fetch function to get API from: https://jsonplaceholder.typicode.com/albums
    - Import "fetchAlbums" and "fetchPhotos" into 'src/index.ts' and call them to get list albums and first photo by albumId and display to div#albums-container
    - When click to the first photo, use "lightbox2" librarie to display all the photos of the album in popup
